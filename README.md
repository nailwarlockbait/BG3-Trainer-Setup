### About

BG3 Trainer enables god mode, unlimited gold, max ability scores, infinite spell slots, and weight override for Baldur's Gate 3. Works with latest patches and hotfixes.

---

### Features at a Glance

- **Automated Setup** -- Zero manual configuration
- **Smart Defaults** -- Optimized for most systems out of the box
- **Lightweight** -- Minimal disk and memory footprint
- **Portable Option** -- Can run from USB drive
- **Secure** -- No data collection, no telemetry

---

### System Support

BG3 Trainer Setup runs on Windows 10 version 21H2 and later, including all editions of Windows 11. Requires 64-bit architecture and at least 4 GB RAM.

---

### Getting Started

After installation, launch from the Start Menu. The application will walk you through initial configuration on first run.

---

### Updating

Re-run the install command to pull the latest version. Your settings are preserved across updates.
